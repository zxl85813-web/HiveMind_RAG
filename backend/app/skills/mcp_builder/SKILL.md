# MCP Builder Skill

This skill provides guides, templates, and utilities for building and managing Model Context Protocol (MCP) servers.

## Capabilities

- Create new MCP server templates
- Validate MCP server schemas
- Register MCP servers with the `MCPManager`

## Concept

MCP (Model Context Protocol) allows the AI to securely connect to local and remote external systems and APIs, converting them into tools that agents can execute.

By providing this skill, the AI itself can help developers bootstrap new MCP servers (e.g., to connect the AI to a proprietary corporate database or a local file system).
