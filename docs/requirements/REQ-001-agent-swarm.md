## 需求描述

系统需要支持多 Agent 协作的蜂巢模式。一个 Supervisor Agent 负责理解用户意图并路由到专业 Agent，多个 Agent 可以并行或串行协作完成复杂任务。

## 详细要求

1. **Supervisor 调度** — 自动分析用户意图，路由到最合适的 Agent
2. **多 Agent 协作** — 支持多个 Agent 协同处理一个请求
3. **Agent 注册/注销** — 支持动态注册新 Agent，热插拔
4. **执行可视化** — 前端可实时展示 Agent 执行过程 (Agent Trace)
5. **基于 LangGraph** — 使用 LangGraph StateGraph 实现工作流编排

## 验收标准

- [ ] 至少支持 3 种专业 Agent (RAG / SQL / Web)
- [ ] Supervisor 能正确路由 90%+ 的测试用例
- [ ] 支持 Agent 动态注册和注销
- [ ] 前端能展示 Agent 执行追踪
