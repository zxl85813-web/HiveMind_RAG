## 需求描述

Agent 集群拥有共享记忆和 TODO List，能够进行自省自纠。当 Agent 发现自身能力不足或判断不确定时，主动提醒用户介入。

## 详细要求

### 共享记忆
1. **三层记忆架构** — Working (短期) / Episodic (中期) / Semantic (长期)
2. **记忆检索** — 支持语义检索历史记忆
3. **记忆衰减** — 旧的不重要记忆自动降权或清理

### 共享 TODO List
1. **自动提取** — 从对话中自动识别待办事项
2. **Agent 认领** — Agent 可以认领并执行 TODO 中的任务
3. **优先级管理** — 支持优先级排序

### 自省 (Reflection)
1. **回答质量自评** — Agent 对自己的输出进行质量评估
2. **错误检测与自纠** — 发现矛盾或错误时自动修正
3. **用户介入提醒** — 信心不足时通过 WebSocket 主动通知用户
4. **周期性回顾** — 定期回顾历史，提炼有价值的知识

## 验收标准

- [ ] 记忆能跨会话持久化
- [ ] 语义检索召回率 > 80%
- [ ] 自省能检测出明显的事实错误
- [ ] 低信心时能成功推送用户通知
